## What I'm Building

### [Subly](https://github.com/colehollander10-netizen/Subly) — iOS free-trial alarm
> *"Know before your trials charge you — without linking your bank."*

Every competitor (Rocket Money, Truebill, Copilot) requires Plaid. Subly reads your Gmail instead. Privacy-first, local-first, SwiftUI + SwiftData + Gmail API. Currently in active development on `trial-only-rebuild`.

**Stack:** Swift · SwiftUI · SwiftData · Gmail REST API · GoogleSignIn · BGAppRefreshTask

---

### [Canvas Student MCP Server](https://github.com/colehollander10-netizen/canvas-student-mcp-server) — Claude ↔ Canvas LMS
> MCP server that gives Claude access to your grades, assignments, files, and announcements.

Built this because I wanted Claude to actually know what was due — not just tell me to check the syllabus. Published and open source.

**Stack:** Node.js · Model Context Protocol · Canvas LMS API

## How I Work

I build with **Claude Code** as a peer engineer, not a code autocomplete. Every project runs on a structured workflow: Linear for issues, Obsidian for notes, Warp for terminal. I treat AI as leverage — not a shortcut around understanding the code.

```
Cursor   → SwiftUI / boilerplate
Claude Code → parser logic, architecture, coordinator
Obsidian → project memory + session notes
Linear   → single source of truth for tasks
```
